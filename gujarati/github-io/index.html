  • Installation and Account Creating

  • Open account on github.com
    • Don't make to long username try make it simple and short
    • jimishf (short and good) and dhruvsaidava (easy but long)
  • Now download git bash for windows from https://git-scm.com/downloads
  • After installation open git bash and check with command git
  • If you get more git command displayed your installation is done properly

  • Creating new repository

After Installation we need to create repository in which we can put file and work on projects.

  • You can create new repo by clicking on + icon situated at right top corner you will get option of new repo
  • now you need to create your first github.io page
  • Name repo "your-username.github.io"
    • example: jimishf.github.io or dhruvsaidava.github.io
    • it should be your username

Git clone
You can clone any repository from git by using Git Clone Command.
Using CD commands get to folder were you want to clone repository.

Git Status
Checking status of files in master

Git add .
To add your all file and ready for commit.

Git Commit -m "Your Comment"
Commit create packet to send with your comment what is change done in to the packet so it shows in timeline

Git Pull
Before push it is advised to do a pull so any changes done by other comes to your local repository.
after pull you may require to use :q command to quit pull request at end.

Git Push
Git Push is final command to send packet to server were it set all file with changes
